# Agent Run Panel

A React + TypeScript application that simulates an agent-based workflow using an event-driven architecture.

## Features

- Event-driven state management (Reducer-based)
- Real-time simulation using mock event emitter
- Task lifecycle visualization (pending → running → complete/failed/cancelled)
- Parallel task execution and dependencies
- Streaming partial outputs
- Tool call tracking
- Retry and failure handling
- Success and error scenarios

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## How to Run

```bash
npm install
npm run dev
